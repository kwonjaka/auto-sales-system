## SCR-01 로그인

### 화면 개요
시스템 접속을 위한 인증 화면

### 레이아웃

```
┌─────────────────────────────────┐
│                                 │
│       영업일일보고 시스템        │
│                                 │
│   ┌─────────────────────────┐   │
│   │ 이메일                  │   │
│   └─────────────────────────┘   │
│   ┌─────────────────────────┐   │
│   │ 비밀번호                │   │
│   └─────────────────────────┘   │
│                                 │
│   [        로그인         ]     │
│                                 │
└─────────────────────────────────┘
```

### 입력 항목

| 항목 | 타입 | 필수 | 유효성 검사 |
|------|------|------|-------------|
| 이메일 | text (email) | Y | 이메일 형식 |
| 비밀번호 | password | Y | 최소 8자 |

### 동작

| 이벤트 | 동작 |
|--------|------|
| 로그인 버튼 클릭 | 인증 후 보고서 목록(SCR-02)으로 이동 |
| 인증 실패 | 에러 메시지 표시 ("이메일 또는 비밀번호가 올바르지 않습니다") |

## SCR-02 보고서 목록

### 화면 개요
일일보고서 목록을 조회하는 화면. 영업사원은 본인 보고서만, 상급자는 팀 전체 보고서를 조회한다.

### 레이아웃

```
┌──────────────────────────────────────────────────────────┐
│ 영업일일보고 시스템                  [홍길동] [로그아웃] │
├──────────────────────────────────────────────────────────┤
│ 보고서 목록                                              │
│                                                          │
│ 기간: [2026-04-01] ~ [2026-04-30]                       │
│ 영업사원: [전체 ▼]          [검색]  [+ 새 보고서 작성]  │
│                                                          │
│ ┌──────┬────────────┬────────┬────────┬───────────────┐  │
│ │ 날짜 │ 영업사원   │방문건수│댓글수  │               │  │
│ ├──────┼────────────┼────────┼────────┼───────────────┤  │
│ │04/25 │ 홍길동     │  3건   │  1건   │ [상세보기]    │  │
│ │04/24 │ 홍길동     │  2건   │  0건   │ [상세보기]    │  │
│ │04/23 │ 홍길동     │  4건   │  2건   │ [상세보기]    │  │
│ └──────┴────────────┴────────┴────────┴───────────────┘  │
│                                                          │
│                    < 1 2 3 ... >                         │
└──────────────────────────────────────────────────────────┘
```

### 검색 조건

| 항목 | 타입 | 설명 |
|------|------|------|
| 기간 (시작) | date | 기본값: 당월 1일 |
| 기간 (종료) | date | 기본값: 오늘 |
| 영업사원 | select | 상급자만 표시. 본인 팀원 목록 |

### 목록 컬럼

| 컬럼 | 설명 |
|------|------|
| 날짜 | 보고서 작성 날짜 |
| 영업사원 | 작성자 이름 |
| 방문건수 | 당일 방문기록 행 수 |
| 댓글수 | 달린 댓글 수 |

### 동작

| 이벤트 | 동작 |
|--------|------|
| 새 보고서 작성 버튼 | SCR-03으로 이동 |
| 상세보기 버튼 | SCR-04로 이동 |
| 당일 보고서 이미 존재 | 새 보고서 작성 버튼 비활성화 |

## SCR-03 보고서 작성

### 화면 개요
영업사원이 당일 일일보고를 작성하는 화면. 하루 1건만 작성 가능.

### 레이아웃

```
┌──────────────────────────────────────────────────────────┐
│ 영업일일보고 시스템                  [홍길동] [로그아웃] │
├──────────────────────────────────────────────────────────┤
│ 보고서 작성                                              │
│                                                          │
│ 작성자: 홍길동          보고일: 2026-04-25               │
│                                                          │
│ ── 방문 기록 ────────────────────────────────────────── │
│ ┌──────────────────────┬─────────────────────┬────────┐  │
│ │ 고객명               │ 방문내용            │        │  │
│ ├──────────────────────┼─────────────────────┼────────┤  │
│ │ [고객 선택 ▼]        │ [방문내용 입력    ] │ [삭제] │  │
│ │ [고객 선택 ▼]        │ [방문내용 입력    ] │ [삭제] │  │
│ └──────────────────────┴─────────────────────┴────────┘  │
│                              [+ 행 추가]                 │
│                                                          │
│ ── 현재 과제 / 상담 ─────────────────────────────────── │
│ ┌────────────────────────────────────────────────────┐   │
│ │                                                    │   │
│ │  (자유 입력)                                       │   │
│ │                                                    │   │
│ └────────────────────────────────────────────────────┘   │
│                                                          │
│ ── 내일 할 일 ───────────────────────────────────────── │
│ ┌────────────────────────────────────────────────────┐   │
│ │                                                    │   │
│ │  (자유 입력)                                       │   │
│ │                                                    │   │
│ └────────────────────────────────────────────────────┘   │
│                                                          │
│                        [취소]  [저장]                    │
└──────────────────────────────────────────────────────────┘
```

### 입력 항목

| 항목 | 타입 | 필수 | 유효성 검사 |
|------|------|------|-------------|
| 고객명 (방문기록) | select | Y | 고객 마스터에서 선택 |
| 방문내용 (방문기록) | textarea | Y | 최대 1000자 |
| 현재 과제/상담 | textarea | N | 최대 2000자 |
| 내일 할 일 | textarea | N | 최대 2000자 |

### 동작

| 이벤트 | 동작 |
|--------|------|
| 행 추가 버튼 | 방문기록 행 1줄 추가 (최대 20행) |
| 삭제 버튼 | 해당 방문기록 행 삭제 |
| 저장 버튼 | 유효성 검사 후 저장, SCR-04로 이동 |
| 취소 버튼 | SCR-02로 이동 (변경사항 있을 경우 확인 팝업) |

## SCR-04 보고서 상세

### 화면 개요
작성된 일일보고를 조회하고, 상급자가 댓글을 작성하는 화면.

### 레이아웃

```
┌──────────────────────────────────────────────────────────┐
│ 영업일일보고 시스템                  [홍길동] [로그아웃] │
├──────────────────────────────────────────────────────────┤
│ 보고서 상세                          [목록] [수정] [삭제]│
│                                                          │
│ 작성자: 홍길동          보고일: 2026-04-25               │
│                                                          │
│ ── 방문 기록 ────────────────────────────────────────── │
│ ┌────┬──────────────────────┬─────────────────────────┐  │
│ │ # │ 고객명               │ 방문내용                │  │
│ ├────┼──────────────────────┼─────────────────────────┤  │
│ │ 1  │ (주)ABC상사          │ 신규 계약 논의          │  │
│ │ 2  │ DEF전자              │ 클레임 대응 방문        │  │
│ └────┴──────────────────────┴─────────────────────────┘  │
│                                                          │
│ ── 현재 과제 / 상담 ─────────────────────────────────── │
│   ABC상사 계약 조건 협의 중. 다음 주 최종 답변 예정.    │
│                                                          │
│ ── 내일 할 일 ───────────────────────────────────────── │
│   GHI물산 방문 및 견적서 제출                           │
│                                                          │
│ ── 댓글 ─────────────────────────────────────────────── │
│ ┌────────────────────────────────────────────────────┐   │
│ │ 김상사 | 2026-04-25 18:30                          │   │
│ │ ABC상사 계약 관련 본부장 보고 필요. 확인 바람.      │   │
│ └────────────────────────────────────────────────────┘   │
│                                                          │
│ ┌────────────────────────────────────────────────────┐   │
│ │ 댓글 입력 (상급자만 작성 가능)                     │   │
│ └────────────────────────────────────────────────────┘   │
│                                      [댓글 등록]         │
└──────────────────────────────────────────────────────────┘
```

### 표시 항목

| 항목 | 설명 |
|------|------|
| 작성자 | 영업사원 이름 |
| 보고일 | 보고서 날짜 |
| 방문기록 | 고객명 + 방문내용 목록 |
| 현재 과제/상담 | 텍스트 |
| 내일 할 일 | 텍스트 |
| 댓글 | 작성자명, 작성일시, 내용 |

### 동작

| 이벤트 | 동작 | 권한 |
|--------|------|------|
| 수정 버튼 | SCR-03(수정 모드)으로 이동 | 본인(당일만) |
| 삭제 버튼 | 확인 팝업 후 삭제, SCR-02로 이동 | 본인(당일만) |
| 댓글 등록 버튼 | 댓글 저장 후 화면 갱신 | 상급자만 |

### 레이아웃

```
┌──────────────────────────────────────────────────────────┐
│ 영업일일보고 시스템                  [홍길동] [로그아웃] │
├──────────────────────────────────────────────────────────┤
│ 고객 등록                                                │
│                                                          │
│ 회사명 *   [                              ]              │
```

## SCR-08 영업사원 마스터 등록/수정

### 화면 개요
영업사원 정보를 등록하거나 수정하는 화면. 상급자만 접근 가능.

### 레이아웃

```
┌──────────────────────────────────────────────────────────┐
│ 영업일일보고 시스템                  [홍길동] [로그아웃] │
├──────────────────────────────────────────────────────────┤
│ 영업사원 등록                                            │
│                                                          │
│ 이름 *     [                              ]              │
│ 이메일 *   [                              ]              │
│ 비밀번호 * [                              ]              │
│ 부서       [                              ]              │
│ 직급       [                              ]              │
│ 상급자     [선택 ▼                        ]              │
│                                                          │
│                        [취소]  [저장]                    │
└──────────────────────────────────────────────────────────┘
```

### 입력 항목

| 항목 | 타입 | 필수 | 유효성 검사 |
|------|------|------|-------------|
| 이름 | text | Y | 최대 50자 |
| 이메일 | text | Y | 이메일 형식, 중복 불가 |
| 비밀번호 | password | Y (신규) | 최소 8자 |
| 부서 | text | N | 최대 50자 |
| 직급 | text | N | 최대 30자 |
| 상급자 | select | N | 영업사원 목록에서 선택 |

### 동작

| 이벤트 | 동작 |
|--------|------|
| 저장 버튼 | 유효성 검사 후 저장, SCR-07로 이동 |
| 취소 버튼 | SCR-07로 이동 |
